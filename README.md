# PDFab Website

This repository hosts the public landing page for **PDFab**, an Android PDF utility app focused on fast, practical document work.

## Included Files

- `index.html` — main landing page
- `app-ads.txt` — AdMob app-ads configuration
- `pdfab-home-screen.jpeg` — home screen image
- `pdfab-tools-screen.jpeg` — tools screen image
- `pdfab-files-screen.jpeg` — files screen image

## App Overview

PDFab is designed for users who need to handle PDFs quickly on mobile. The app focuses on core document tasks such as:

- Merging PDFs
- Splitting PDFs
- Converting PDF to JPG
- Converting JPG to PDF
- Compressing PDF files
- Scanning documents to PDF
- Signing PDF files
- Extracting text from PDFs and scans
- Managing files on-device
